# Mariyamma Murshida — Portfolio

A personal portfolio website built with a **minimalist-brutalist** design aesthetic, featuring bold typography, clean layouts, and smooth scroll animations.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

---

## ✨ Features

- **Hero Section** — Bold name display with outline text effect, designation, and CTA buttons
- **About Me** — Bio with animated stats (Projects, Technologies, Curiosity)
- **Experience** — Timeline card with role, company, and responsibilities
- **Skills & Technologies** — Categorized grid cards (Frontend, Backend, Mobile, Tools)
- **Projects** — Showcase cards with tech stack tags and live links
- **Contact** — Contact info links + functional contact form (mailto)
- **Footer** — Compact bar with copyright and social icons

## 🎨 Design

- Brutalist-minimalist aesthetic inspired by [studio.site](https://preview.studio.site/templates/gBRO381qDm/)
- Light gray palette (`#f2f0ed`) with black typography
- **Fonts**: Space Grotesk (headings) + Inter (body) via Google Fonts
- Scroll reveal animations using Intersection Observer API
- Fully responsive (desktop, tablet, mobile)
- Smooth hover micro-animations on cards, tags, and links

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic page structure |
| CSS3 | Custom properties, Grid, Flexbox, animations |
| JavaScript | Scroll animations, mobile nav, form handling |

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/murshida-123/PORTFOLIO.git
   ```

2. **Open locally**
   ```bash
   cd PORTFOLIO
   ```
   Open `index.html` in your browser, or use a live server:
   ```bash
   npx serve .
   ```

3. **View at** `http://localhost:3000`
